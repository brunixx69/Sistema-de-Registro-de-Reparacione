# 🛠️ FixMaster - Sistema de Gestión de Reparaciones

FixMaster es una aplicación web profesional diseñada para servicios técnicos que necesitan gestionar órdenes de trabajo, diagnósticos y estados de reparación en tiempo real.

## 🚀 Características Principales

- **Dashboard de Métricas:** Visualización en tiempo real de órdenes totales, pendientes, en proceso y finalizadas.
- **Ciclo de Vida de Reparación:** Registro inicial del cliente y actualización posterior del diagnóstico profesional.
- **Gestión Completa (CRUD):** Crear, editar, visualizar y eliminar registros con persistencia de datos.
- **Diseño Ultra-Responsivo:** Interfaz adaptativa (Mobile-First) que transforma tablas en tarjetas para una mejor experiencia en celulares y tablets.
- **Modo Oscuro/Claro:** Interfaz ergonómica con persistencia de preferencia mediante `localStorage`.
- **Buscador Inteligente:** Filtro rápido por código de orden o nombre del cliente.

## 🛠️ Tecnologías Utilizadas

- **HTML5:** Estructura semántica para SEO y accesibilidad.
- **CSS3:** Variables modernas, Grid, Flexbox y animaciones avanzadas (Glassmorphism).
- **JavaScript (ES6+):** Manipulación avanzada del DOM, Fetch API, LocalStorage y lógica funcional.

## 📱 Vista Previa en Móvil

La aplicación ha sido optimizada para dispositivos táctiles, eliminando el "delay" de clic y adaptando los formularios para un uso fluido con el pulgar.
